## About KineticButton

KineticButton are composed of text inside a `span` element and may include inline/inline-block elements and/or icons.

### Colors

There are six different color kinetic-button you can use.

| KineticButton | class | description |
| ----- | ----- | ----------- |
| <KineticButton key="1">Default KineticButton</KineticButton>| `.slds-button_kinetic`| This is the default kinetic-button. You probably want to use this one unless you fall into a specific use-case.|
| <InverseKineticButton key="2">Darker KineticButton</InverseKineticButton>| `.slds-button_kinetic_inverse`| This kinetic-button is used when you need higher contrast than our default kinetic-button.|
| <LightestKineticButton key="3">Lightest KineticButton</LightestKineticButton>| `.slds-button_kinetic_lightest`| This kinetic-button is used when you need higher contrast, such as on colored backgrounds.|
| <KineticButton key="4" isSuccess>Success KineticButton</KineticButton>| `.slds-button_kinetic .slds-theme_success`| This is a kinetic-button used to communicate success.|
| <KineticButton key="5" isWarning>Warning KineticButton</KineticButton>| `.slds-button_kinetic .slds-theme_warning`| This is a kinetic-button used to communicate a warning.|
| <KineticButton key="6" isError>Error KineticButton</KineticButton>| `.slds-button_kinetic .slds-theme_error`| This is a kinetic-button used to communicate an error.|

## KineticButton with Icons

You can include an icon in your kinetic-button. For maximum visual clarity, [Utility icons](/icons/#utility) are the suggested icon type inside kinetic-button. You may also choose to put an icon in a kinetic-button without text, but in that case, be sure to include assistive text.

The icon can be aligned to the left or right side of the text. To use an icon, wrap the icon with `<span class="slds-button_kinetic__icon">`. Add the classes `slds-button_kinetic__icon_left` or `slds-button_kinetic__icon_right` to the wrapper for spacing the icon in relation to the text label.

## KineticButton with Nested Elements

Along with icons, you can have nested textual elements, such as `strong`, `em`, and `span` elements, within a kinetic-button. Only `inline` and `inline-block` elements are supported at this time.

## About AKineticsComponent

AKineticsComponent are composed of text inside a `span` element and may include inline/inline-block elements and/or icons.

### Colors

There are six different color a-kinetics-component you can use.

| AKineticsComponent | class | description |
| ----- | ----- | ----------- |
| <AKineticsComponent key="1">Default AKineticsComponent</AKineticsComponent>| `.slds-a-kinetics-component`| This is the default a-kinetics-component. You probably want to use this one unless you fall into a specific use-case.|
| <InverseAKineticsComponent key="2">Darker AKineticsComponent</InverseAKineticsComponent>| `.slds-a-kinetics-component_inverse`| This a-kinetics-component is used when you need higher contrast than our default a-kinetics-component.|
| <LightestAKineticsComponent key="3">Lightest AKineticsComponent</LightestAKineticsComponent>| `.slds-a-kinetics-component_lightest`| This a-kinetics-component is used when you need higher contrast, such as on colored backgrounds.|
| <AKineticsComponent key="4" isSuccess>Success AKineticsComponent</AKineticsComponent>| `.slds-a-kinetics-component .slds-theme_success`| This is a a-kinetics-component used to communicate success.|
| <AKineticsComponent key="5" isWarning>Warning AKineticsComponent</AKineticsComponent>| `.slds-a-kinetics-component .slds-theme_warning`| This is a a-kinetics-component used to communicate a warning.|
| <AKineticsComponent key="6" isError>Error AKineticsComponent</AKineticsComponent>| `.slds-a-kinetics-component .slds-theme_error`| This is a a-kinetics-component used to communicate an error.|

## AKineticsComponent with Icons

You can include an icon in your a-kinetics-component. For maximum visual clarity, [Utility icons](/icons/#utility) are the suggested icon type inside a-kinetics-component. You may also choose to put an icon in a a-kinetics-component without text, but in that case, be sure to include assistive text.

The icon can be aligned to the left or right side of the text. To use an icon, wrap the icon with `<span class="slds-a-kinetics-component__icon">`. Add the classes `slds-a-kinetics-component__icon_left` or `slds-a-kinetics-component__icon_right` to the wrapper for spacing the icon in relation to the text label.

## AKineticsComponent with Nested Elements

Along with icons, you can have nested textual elements, such as `strong`, `em`, and `span` elements, within a a-kinetics-component. Only `inline` and `inline-block` elements are supported at this time.
